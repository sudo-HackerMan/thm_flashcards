======
Инструкция по установке
  На Windows:
    1. Установить шрифт Ubuntu-R.ttf из директории win32
      (ПКМ => Установить шрифт)
    2. Запустить "Карточки_setup.exe" из директории win32
      и проследовать шагам установщика
    3. Запустить ярлык "Карточки" на рабочем столе
  На Linux:
    1. Проверить, что Python 2.7.18 установлен
    2. Запустить файл "linux_setup.py" из директории linux
       от имени пользователя root и проследовать шагам установщика
       (sudo python2 linux_setup.py)
    3. В меню приложений, категория Игры вашего окружения рабочего
       стола открыть ярлык "Карточки". По желанию вынести на рабочий 
       стол
======
Минимальные системные требования:
  ОС: Windows 7 / Linux (любой, куда можно поставить Python 3.8)
  Процессор: 1 ядро, 1 поток
  ОЗУ: 1 гБ (включая ОС)
  Разрешение экрана: 800х600
  (для Linux): наличие Python 2.7.18.

Рекомендуемые системные требования:
  ОС: Windows 7 / Linux (любой, куда можно поставить Python 3.8)
  Процессор: 2 ядра, 4 потока
  ОЗУ: 2 гБ (включая ОС)
  Разрешение экрана: 1024х768
  (для Linux): наличие Python 2.7.18.
======
